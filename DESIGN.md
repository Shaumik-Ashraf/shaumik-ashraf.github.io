# Design

## User Story

A user visits the website and is greeted by a 2D avatar sprite with intuitive WASD controls and movement. The main section of the screen is a 2D side-scrolling platformer. Platforms may spawn in for creativity in movement. Gaps may spawn in which the user must react to or else restart. Stars/dots will spawn in, and when the avatar hits these items the page will redirect to that website. Entities will slide in from right to left. The navigation bar will have restart/pause/settings options, effectively doubling as a menu for the game. 

## User Persona

Someone already familiar with programming and the web, as this website is already hosted on GitHub. A geek or nerd.

## Requirements

- SPA that fully renders with GitHub pages.
- Possible website redirects are defined in a centralized JSON file.
